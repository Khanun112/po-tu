ในไฟล์ po621.html ข้อมูลเอกสารทุกรายการอยู่ใน array ชื่อ readings แต่ละรายการมี field ชื่อ link — ถ้า field นั้นมี URL เว็บจะแสดงปุ่ม "เปิดเอกสาร" ออกมาให้คลิกได้ทันที

⚙️ ขั้นตอน: เพิ่ม URL ให้เอกสาร

1
เปิดไฟล์ po621.html ด้วย text editor (แนะนำ VS Code หรือ Notepad++)
2
กด Ctrl+F (หรือ Cmd+F บน Mac) แล้วค้นหาชื่อเอกสารที่ต้องการ เช่น Toppling Thaksin
3
หา field linkEN:"" และ linkTH:"" ในบรรทัดนั้น แล้วใส่ URL ที่ต้องการ
ใส่แค่ภาษาเดียวก็ได้ — ระบบจะแสดงเฉพาะปุ่มที่มี URL เท่านั้น

ตัวอย่าง — มีแค่ EN:
linkEN:"https://newleftreview.org/...", linkTH:""

ตัวอย่าง — มีทั้งสองภาษา:
linkEN:"https://...", linkTH:"https://drive.google.com/..."
4
บันทึกไฟล์ → อัปโหลดไฟล์ใหม่ขึ้น GitHub (drag & drop แทนไฟล์เดิม) → รอ 1 นาที เว็บอัปเดตทันที

📂 แหล่งหา URL เอกสาร

A
Google Scholar — ค้นชื่อบทความ → คลิก PDF หรือ View → copy URL
B
ResearchGate / Academia.edu — หนังสือและบทความจำนวนมากมี PDF ฟรี
C
Google Drive ของตัวเอง — อัปโหลด PDF → Share → "Anyone with link" → copy link นั้น
วิธีนี้เหมาะกับเอกสารที่ดาวน์โหลดมาแล้วหรือเอกสารส่วนตัว
D
JSTOR / Taylor & Francis — บทความในวารสาร ใช้ DOI link ได้เลย เช่น https://doi.org/10.1080/...
